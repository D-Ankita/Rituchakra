# Supabase vs Firebase vs Convex

**Status:** research note · stack recommendation basis

**Supabase** — Postgres + RLS + predictable pricing + open-source + ap-south-1 (Mumbai) region. Best fit for DPDP-compliant India launch. **Firebase** — mature but per-read pricing balloons at scale · no India region (Singapore closest). **Convex** — excellent DX for real-time · no India region · newer. **Decision: Supabase** for Mumbai region + SQL power + RLS.
